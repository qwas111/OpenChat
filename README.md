## **一、产品简介**  
**OpenChat** 是一款专注于隐私保护的即时通讯软件，采用端到端加密（E2EE）技术，确保只有对话双方能读取消息内容。无需手机号注册，无广告追踪，无云端数据存储——**你的对话，只属于你**。


## **二、核心功能亮点**  
### 🔒 **1. 端到端加密（E2EE）**  
- 所有消息、文件、语音通话在发送前自动使用**非对称**密钥加密，私钥密钥仅存于用户设备，**连 我们 也无法访问**。  
- 支持 **“前向保密”** 技术：每次会话使用独立密钥，即使单次密钥泄露，历史消息仍安全。  


### 🚫 **3. 零元数据留存**  
- 不收集手机号/邮箱（仅需用户名注册），不存储联系人关系、聊天时间戳等元数据。  
- 消息可以阅后即焚，彻底抹除隐私痕迹。  


### ⚡ **4. 轻量化跨平台体验**  
- 支持 **Windows、Linux（不包括Android，仍在制作）、Android（公测中）、macOS**，全平台无缝同步。  
- 低带宽优化：在弱网环境下仍可发送消息与文件。  


### ✨ **5. 用户友好设计**  
- **清爽无广告界面**：专注沟通本质，无干扰信息流。

## 技术栈

*   **加密技术**:
    *   **X25519**: 用于生成密钥对和进行密钥交换，以派生共享密钥。
    *   **Ed25519**: 用于生成密钥对和进行消息签名与验证。
    *   **AES (CBC 模式)**: 使用派生的共享密钥对消息进行对称加密和解密。
    *   **PKCS7 填充**: 用于 AES 加密前的消息填充。
    *   **HKDF (SHA256)**: 用于从共享密钥派生出 AES 密钥。
*   **后端**:
    *   **编程语言**: Python
    *   **框架**: Flask
    *   **数据库**: SQLite
    *   **速率限制**: Flask-Limiter
    *   **服务器**: Waitress
*   **前端**:
    *   **编程语言**: Python
    *   **GUI 框架**: PyQt5
    *   **网络请求**: aiohttp
    *   **异步处理**: asyncio, qasync

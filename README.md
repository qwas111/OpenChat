# OpenChat 加密聊天项目

## 简介

OpenChat 是一个致力于提供安全、私密的即时通讯体验的开源项目。我们相信通信的隐私权是一项基本人权，因此我们致力于构建一个强大且易于使用的加密聊天应用程序。

## 特点

*   **端到端加密**: 您的所有消息都经过端到端加密，这意味着只有您和您通信的对象才能读取消息内容。即使是 OpenChat 的开发者也无法访问您的对话。
*   **安全即时通讯**: 提供稳定可靠的即时消息发送和接收功能，确保您的沟通畅通无阻。
*   **注重隐私**: 我们将用户隐私置于首位，不收集不必要的个人信息，并采取严格的安全措施保护您的数据。

## 目标用户

OpenChat 主要面向注重隐私和数据安全的个人用户。如果您希望在与朋友、家人或同事交流时，确保对话的私密性，OpenChat 将是您的理想选择。

## 技术栈

*   **加密技术**:
    *   **X25519**: 用于生成密钥对和进行密钥交换，以派生共享密钥。
    *   **Ed25519**: 用于生成密钥对和进行消息签名与验证。
    *   **AES (CBC 模式)**: 使用派生的共享密钥对消息进行对称加密和解密。
    *   **PKCS7 填充**: 用于 AES 加密前的消息填充。
    *   **HKDF (SHA256)**: 用于从共享密钥派生出 AES 密钥。
*   **后端**:
    *   **编程语言**: Python
    *   **框架**: Flask
    *   **数据库**: SQLite
    *   **速率限制**: Flask-Limiter
    *   **服务器**: Waitress
*   **前端**:
    *   **编程语言**: Python
    *   **GUI 框架**: PyQt5
    *   **网络请求**: aiohttp
    *   **异步处理**: asyncio, qasync
